# widget-launcher
The tool that launches command-line widgets.

Okay, you've read this bit ↑, that is mostly all, but you would probably say "How would i make my own widgets?" or "What would i do?" well, we have answers.
Before doing any of this, you may or may not have installed python, if you haven't, download it from here. "https://www.python.org/getit/windows/" So now just by installing that, you're set!
